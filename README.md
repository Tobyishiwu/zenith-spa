# 🌿 Zenith Spa

A modern luxury mobile spa booking platform built with the MERN stack.

Zenith Spa enables customers to book premium mobile spa services, upload payment proof, track appointments, and receive booking confirmations, while giving administrators complete control over therapists, bookings, payment methods, and business settings.

------------------------------------------------------------

FEATURES

Customer Portal

- Luxury responsive website
- Browse spa services
- View therapist profiles
- Book appointments
- Upload payment proof
- Track booking status
- Download booking confirmation
- About page
- Contact page

Admin Dashboard

- Dashboard overview
- Booking management
- Therapist management
- Payment method management
- Business settings
- Payment approval workflow

LOCAL SETUP

Backend

cd zenith-spa-server
npm install
npm run dev

Frontend

cd zenith-spa-client
npm install
npm run dev

------------------------------------------------------------

ENVIRONMENT VARIABLES

Create a .env file inside zenith-spa-server

Required variables

PORT=5000

MONGODB_URI=<your_mongodb_connection_string>

JWT_SECRET=<your_jwt_secret>

CLIENT_URL=http://localhost:5173
